# 👟 Sneak Freak – Shoes E-commerce

Sneak Freak is a sleek, frontend-based e-commerce platform dedicated to sneaker enthusiasts. 
This responsive website provides a visually engaging interface for users to browse, explore, 
and view detailed product pages for trending sneakers.

---

## 📁 Project Structure

- `index.html` – Homepage landing with featured sneakers.
- `about.html` – About the store/brand.
- `contact.html` – Contact form or information.
- `blog.html` – Blog or news page for sneaker updates.
- `login.html` – Basic login UI for returning users.
- `shop.html` – Main shop page listing available sneakers.
- `card.html` – Product card layout.
- `style.css` – Main stylesheet for the entire site.
- `index.js` – JavaScript for interactive elements.
- `assest/` – Images, icons, and other media assets.
- `homeprod1.html` to `homeprod8.html` – Individual product showcase pages from homepage.
- `shop1prod.html` to `shop4prod.html` – Product detail pages from the shop section.

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Design Approach**: Responsive layout using custom CSS
- **Assets**: Organized in `assest/` folder

---

## ✨ Key Features

- Fully static ecommerce UI
- Individual product detail pages
- Clean, responsive design
- Organized asset and page structure
- Scrollable and interactive product displays

---

## 🎯 Purpose

To build a niche ecommerce platform tailored to sneaker lovers while learning frontend UI/UX principles and best practices in responsive design.

---

## 🚀 How to Run

1. Download or clone the repository.
2. Open `index.html` in any browser.
3. Navigate through pages using header links or buttons.

> 💡 *Note: As a frontend-only project, there is no backend or database integration.*
